# Artefatos de release

Esta pasta contem os artefatos Windows gerados para distribuicao do PhotoRescue 0.1.0.

## Arquivos

- `PhotoRescue.exe`: executavel direto, sem instalador.
- `PhotoRescue_0.1.0_x64-setup.exe`: instalador Windows NSIS.
- `PhotoRescue_0.1.0_x64_en-US.msi`: instalador Windows MSI.
- `checksums.sha256`: hashes SHA-256 dos artefatos.

## Observacao

O PhotoRescue ainda esta em desenvolvimento. Use com cuidado, execute preferencialmente como administrador e nunca salve arquivos recuperados na mesma unidade escaneada.
